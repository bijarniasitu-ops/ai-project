
# ResumeRanker 🏆

AI-powered resume screening tool for smarter hiring decisions.

## 🧠 What it does
Upload multiple resumes and a job description — ResumeRanker ranks candidates by relevance using NLP.

## ⚙️ Project Workflow
1. User uploads multiple resumes (PDF format)
2. User enters job description
3. System extracts text from resumes
4. AI model analyzes each resume
5. Resumes are scored based on relevance
6. Results displayed in ranked order with insights

## 🚀 Features
- Resume parsing (PDF + DOCX)
- NLP-based ranking using TF-IDF
- Skill extraction
- Match score with explanation
- Clean and responsive UI

## 🧰 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI, Python
- Database: MongoDB
- NLP: spaCy, scikit-learn
